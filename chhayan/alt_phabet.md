Double vowels make these sounds:
| Vowel | Doubled sound |
| - | - |
| a | aah |
| i | eeh |
| u | ooh like in shoot |

Consonants with an apostrophe after make these sounds:
| Consonant | Sound |
| - | - |
| s | sh |
| z | zh |
| t | th |
| d | Other th |
| r | Rolled r |
| l | L with touching your top teeth |
