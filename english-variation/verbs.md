# Verbs
Conjugations:
| | Singular | Plural |
| --- | --- | --- |
| 1st person | -e | -es |
| 2nd person | -a | -as |
| 3rd person | -i | -is |

Examples:
| Normal English | Variation |
| --- | --- |
| I sing | I singe |
| We jump | We jumpes |
| You play(one person) | You playa |
| You play(multiple people) | Youst playas |
| He runs | Heest runi |
| She runs | Sheest runi |
| They hear | They hearis |
